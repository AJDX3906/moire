## **AI Agent for Security **  
  


  


AI 应用于安全，其实和其他场景类似，要认识到 AI 的边界在哪里，哪些事能干，哪些事得加工具、加提示词、加约束才能干，这点很重要，只能从实践中学到。

  


在这个边界的基础认知上，用现有的通用 Agent 实现技巧结合渗透测试思路去拓宽 AI 的边界，配工具、配 memory、配知识库… …

  


对于一个开源作品，除了一个强劲的内核，易用性占非常的大的比重，界面好不好看、配置繁不繁琐，尽可能开箱即用，毕竟爱折腾的人直接用 claude code 了，干嘛费劲安装依赖去试用你的作品。
